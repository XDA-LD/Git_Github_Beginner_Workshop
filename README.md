# 📖 Lab Overview: Git/GitHub Workshop

**Host:** LAU Byblos Software Engineering Club  
**Coordinator:** Rami Kronbi

### 🎯 Purpose

The "Git Training Gauntlet" is designed to move students from theoretical knowledge to practical confidence. It simulates the high-pressure environment of a real software team, covering everything from basic commits to complex rebases and conflict resolution.

### 🏗️ File Structure

The repository starts with a minimal footprint to keep focus on the `.git` directory:

- `app.js`: The primary logic file (used for merge conflicts).
- `config.txt`: A configuration file (used for stashing).
- `notes/todo.txt`: A simple text tracker (used for history and resets).
- `.gitignore`: (To be created by the student) manages untracked files.

### 🛠️ Workshop Flow

1. **The Setup:** Tooling and identity configuration.
2. **The Loop:** Branching, editing, and committing.
3. **The Crisis:** Handling conflicts and interruptions.
4. **The Cleanup:** Amending history and rebasing for a clean timeline.
